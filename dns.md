# DNS (Domain Name System)

DNS translates domain names into IP addresses.

Example:

```txt
google.com → 142.250.x.x
```

Without DNS, users would need to remember IP addresses instead of website names.

---

# How DNS Works

1. A user enters a domain name
2. The device sends a DNS request
3. A DNS server responds with the IP address
4. The device connects to the destination server

---

# Common DNS Port

| Port | Protocol |
|------|-----------|
| 53 | UDP/TCP |

DNS commonly uses UDP for speed.
