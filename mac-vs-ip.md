# MAC Address vs IP Address

MAC addresses and IP addresses are both used in networking, but they serve different purposes.

---

# MAC Address

A MAC (Media Access Control) address is a physical hardware address assigned to a network interface card (NIC).

MAC addresses operate at:

```txt
Layer 2 — Data Link Layer
```

Example:

```txt
00:1A:2B:3C:4D:5E
```

---

# IP Address

An IP address is a logical address used to identify devices on a network.

IP addresses operate at:

```txt
Layer 3 — Network Layer
```

Example:

```txt
192.168.1.10
```

---

# Key Difference

## MAC Address
- Physical identity
- Assigned to hardware
- Used inside local networks

## IP Address
- Logical identity
- Assigned by DHCP or manually
- Used for communication between networks

---

# Real World Example

Think of it like this:

## MAC Address
Like the serial number of a device.

## IP Address
Like the device's current home address.

---

# Why Both Matter

Networking devices use:
- MAC addresses for local communication
- IP addresses for routing traffic across networks
