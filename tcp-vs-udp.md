# TCP vs UDP

TCP and UDP are transport layer protocols used for communication between devices.

---

# TCP (Transmission Control Protocol)

TCP is connection-oriented.

This means:
- devices establish a connection before sending data
- reliability is prioritized

TCP Features:
- error checking
- packet ordering
- retransmission

Common Uses:
- HTTPS
- SSH
- FTP
- Email

---

# UDP (User Datagram Protocol)

UDP is connectionless.

This means:
- data is sent without establishing a connection
- speed is prioritized over reliability

UDP Features:
- faster than TCP
- lower overhead
- no guaranteed delivery

Common Uses:
- Gaming
- VoIP
- Streaming
- DNS

---

# Key Difference

TCP:
- reliable
- slower

UDP:
- faster
- less reliable
