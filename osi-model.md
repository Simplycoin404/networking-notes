# OSI Model

The OSI Model is a framework that explains how devices communicate over a network.

---

# Layer 7 — Application

Provides network services to applications.

Examples:
- HTTP
- HTTPS
- DNS

---

# Layer 6 — Presentation

Handles:
- encryption
- compression
- data formatting

---

# Layer 5 — Session

Manages communication sessions between devices.

---

# Layer 4 — Transport

Handles:
- reliability
- segmentation
- flow control

Protocols:
- TCP
- UDP

---

# Layer 3 — Network

Handles routing and IP addressing.

Devices:
- Routers

Protocol:
- IP

---

# Layer 2 — Data Link

Handles MAC addresses and switching.

Devices:
- Switches

---

# Layer 1 — Physical

Handles physical connections and signals.

Examples:
- Ethernet cables
- Fiber optic cables
