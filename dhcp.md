# DHCP (Dynamic Host Configuration Protocol)

DHCP automatically assigns IP addresses to devices on a network.

Without DHCP, devices would need to be configured manually.

---

# What DHCP Assigns

DHCP can provide:
- IP address
- subnet mask
- default gateway
- DNS server

---

# DHCP Process

The DHCP process is commonly called:

```txt
DORA
```

## D — Discover
The device searches for a DHCP server.

## O — Offer
The DHCP server offers an IP address.

## R — Request
The device requests the offered address.

## A — Acknowledge
The DHCP server confirms the assignment.
